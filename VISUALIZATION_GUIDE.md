# 可视化远程控制系统使用指南

## 概述

本系统提供了一个功能完整的可视化远程控制界面，所有功能都集成在一个窗口中，点击即用。

## 启动方式

### 被控端（服务端）
```bash
python server_gui.py
```

### 控制端（客户端）- 可视化版本
```bash
python client_visual.py
```

## 主要功能模块

### 1. 📊 仪表盘（DASHBOARD）
**点击即用的实时监控面板**

- **系统信息卡片**
  - 操作系统类型
  - 主机名
  - IP 地址
  - CPU 核心数

- **资源使用率可视化**
  - CPU 使用率（带进度条和百分比显示）
  - 内存使用率（显示已用/总容量）
  - 磁盘使用率（显示已用/总容量）
  - 进度条根据使用率自动变色：
    - 绿色（<60%）：安全
    - 橙色（60-80%）：警告
    - 红色（>80%）：危险

- **实时监控指标**
  - 网络连接数
  - 运行进程数量
  - 系统运行时间
  - 实时日志输出

**特点**：所有数据每3秒自动刷新，无需手动操作

---

### 2. ⚙️ 进程管理器（PROCESSES）
**一键式进程管理**

- **功能按钮**
  - 🔄 **REFRESH**：刷新进程列表
  - 🛑 **KILL PROCESS**：结束选中的进程
  - 搜索框：实时过滤进程

- **进程信息展示**
  - PID（进程ID）
  - 进程名称
  - CPU使用率
  - 内存使用率
  - 运行状态

- **操作方式**
  1. 点击刷新按钮获取最新进程列表
  2. 在列表中选择目标进程
  3. 点击"KILL PROCESS"结束进程
  4. 使用搜索框快速定位进程

---

### 3. 📁 文件浏览器（FILES）
**图形化文件系统管理**

- **路径导航**
  - 地址栏输入路径
  - 双击文件夹进入
  - ".." 返回上级目录

- **文件操作（一键完成）**
  - ⬇️ **Download**：下载选中文件到本地
  - ⬆️ **Upload**：上传本地文件到远程
  - 🗑️ **Delete**：删除选中文件
  - 📝 **View**：在新窗口中查看文件内容

- **文件信息显示**
  - 文件类型（图标）
  - 文件名
  - 文件大小
  - 修改时间

**使用示例**
1. 在路径框输入 `/home/user/documents`
2. 点击 GO 按钮
3. 双击文件夹浏览内容
4. 选择文件后点击下载/查看/删除

---

### 4. 🖥️ 屏幕监控（SCREEN）
**实时屏幕查看**

- **控制面板**
  - ▶ **START STREAM**：开始实时屏幕流
  - ⏹ **STOP STREAM**：停止屏幕流
  - 📸 **SCREENSHOT**：截取单帧截图
  - 质量滑块：调整图像质量（10-100）

- **显示特性**
  - 实时视频流显示在标签页中
  - 自动缩放适应窗口
  - 低延迟传输（可调节FPS）

**操作步骤**
1. 调整质量滑块（默认50，越高越清晰但传输慢）
2. 点击"START STREAM"开始
3. 屏幕画面实时显示
4. 点击"STOP STREAM"停止
5. 或点击"SCREENSHOT"获取单张截图

---

### 5. 💻 Shell终端（SHELL）
**远程命令执行**

- **终端界面**
  - 黑色背景，绿色文字（极客风格）
  - 命令历史记录
  - 实时输出显示

- **使用方式**
  1. 在底部输入框输入命令
  2. 按回车或点击"SEND"发送
  3. 命令输出即时显示在上方
  4. 支持所有配置文件中允许的命令

**示例命令**
```bash
ls -la          # 列出文件
pwd             # 显示当前目录
cat test.txt    # 查看文件内容
ps aux          # 查看进程
```

---

### 6. 📜 操作历史（HISTORY）
**完整的操作日志**

- **记录内容**
  - 时间戳
  - 操作类型
  - 详细信息
  - 状态（成功/失败）

- **功能按钮**
  - 🗑️ **CLEAR**：清空历史记录
  - 💾 **EXPORT**：导出为文本/CSV文件

- **自动记录的操作**
  - 连接/断开
  - 截图/摄像
  - 文件操作
  - 进程管理
  - Shell命令
  - 系统信息查询

**使用场景**
- 审计操作记录
- 追溯问题原因
- 导出操作报告

---

## 底部快捷操作栏

无论在哪个标签页，底部始终显示快捷按钮：

- **📸 Screenshot**：快速截图
- **📷 Camera**：调用摄像头拍照
- **🎤 Mic Record**：录制麦克风音频
- **ℹ️ Sys Info**：查看完整系统信息

点击后自动执行，结果在新窗口显示。

### 实时更新
- 仪表盘数据自动刷新（3秒间隔）
- 屏幕流实时传输（可调FPS）
- 操作历史自动记录

## 连接流程

### 1. 启动服务端
```bash
python server_gui.py
```
- 点击"START SERVICE"开始监听
- 界面显示本机IP和端口
- 等待连接请求

### 2. 启动客户端
```bash
python client_visual.py
```
- 输入目标IP（被控端IP）
- 输入密码（默认：Yssx）
- 点击"ESTABLISH CONNECTION"

### 3. 授权确认
- 被控端弹出授权窗口
- 显示请求来源IP和端口
- 点击"是"允许连接
- 点击"否"拒绝连接

### 4. 开始控制
- 连接成功后进入可视化仪表盘
- 所有功能标签页自动加载
- 后台监控自动启动

---

## 安全提示

1. **密码保护**：修改 `config.py` 中的默认密码
2. **授权机制**：被控端每次连接都需要手动授权
3. **可视化警告**：被控端连接状态栏变红提醒
4. **操作日志**：所有操作都有完整记录
5. **沙箱环境**：文件操作限制在安全目录内

---

## 故障排除

### 连接失败
- 检查IP地址是否正确
- 确认防火墙未阻止端口9999
- 验证密码是否正确

### 功能无响应
- 检查网络连接状态
- 查看操作历史中的错误信息
- 重启客户端/服务端

### 屏幕流卡顿
- 降低质量设置（滑块调低）
- 减少FPS（修改代码中的fps参数）
- 检查网络带宽

---

## 技术架构

- **前端框架**：Tkinter + ttk
- **图像处理**：PIL/Pillow
- **网络通信**：Socket + 自定义协议
- **多线程**：后台监控、流传输独立线程
- **主题系统**：gui_theme.py 统一配置

---

## 扩展开发

### 添加新的监控指标
编辑 `client_visual.py` 中的 `_create_dashboard_tab()` 方法

### 自定义主题颜色
修改 `gui_theme.py` 中的 COLORS 字典

### 添加新的快捷操作
在 `_create_quick_actions()` 中添加按钮
