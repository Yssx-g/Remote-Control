## 📖 简介

这是一个功能完整的**可视化远程控制系统**，控制端采用极客风格的图形界面，集成了6大可视化模块，所有功能都是**点击即用**。

### 🎯 核心特性

- ✅ **实时系统监控**：CPU、内存、磁盘使用率可视化
- ✅ **进程管理器**：图形化查看和管理远程进程
- ✅ **文件浏览器**：可视化文件系统操作
- ✅ **屏幕监控**：实时屏幕流和截图
- ✅ **Shell终端**：远程命令执行
- ✅ **操作历史**：完整的操作日志记录

## 🚀 快速开始

### 1️⃣ 安装依赖

```bash
pip install pillow
```

### 2️⃣ 启动服务端（被控端）

```bash
python server_gui.py
```

点击 **"START SERVICE"** 按钮

### 3️⃣ 启动客户端（控制端）

**方法A：使用快速启动脚本**
```bash
python start_visual_client.py
```

**方法B：直接运行**
```bash
python client_visual.py
```

### 4️⃣ 连接

1. 输入目标IP和密码（默认：`Yssx`）
2. 点击 **"ESTABLISH CONNECTION"**
3. 服务端授权窗口点击 **"是"**
4. 开始使用！

详细教程请查看：[QUICK_START.md](QUICK_START.md)

## 📊 功能详解

### 🎯 仪表盘（自动运行）

**无需操作，自动显示**：
- CPU使用率（进度条+百分比）
- 内存使用率（进度条+已用/总量）
- 磁盘使用率（进度条+已用/总量）
- 网络连接数
- 进程数量
- 系统运行时间
- 实时日志

**特色**：
- 3秒自动刷新
- 智能颜色编码（绿/橙/红）
- 进度条可视化

---

### ⚙️ 进程管理器（点击即用）

**一键操作**：
1. 点 **"REFRESH"** 刷新列表
2. 选择要结束的进程
3. 点 **"KILL PROCESS"** 结束

**显示信息**：
- PID、进程名、CPU%、内存%、状态

---

### 📁 文件浏览器（点击即用）

**操作步骤**：
1. 输入路径，点 **"GO"**
2. 双击文件夹进入
3. 选择文件后：
   - 点 **"Download"** 下载
   - 点 **"View"** 查看
   - 点 **"Delete"** 删除
   - 点 **"Upload"** 上传

**显示信息**：
- 文件类型、名称、大小、修改时间

---

### 🖥️ 屏幕监控（点击即用）

**实时监控**：
1. 调整质量滑块（10-100）
2. 点 **"START STREAM"** 开始
3. 实时画面显示
4. 点 **"STOP STREAM"** 停止

**单帧截图**：
- 点 **"SCREENSHOT"** 立即截图

---

### 💻 Shell终端（点击即用）

**命令执行**：
1. 输入命令（如：`ls -la`）
2. 按回车或点 **"SEND"**
3. 结果即时显示

**支持命令**：
- 文件操作：`ls`, `cd`, `cat`, `cp`, `mv`, `rm`
- 系统信息：`ps`, `top`, `whoami`, `hostname`
- 进程管理：`kill`, `pkill`

---

### 📜 操作历史（自动记录）

**自动记录**：
- 所有操作的时间、类型、详情、状态

**导出功能**：
- 点 **"EXPORT"** 导出为TXT/CSV
- 用于审计和问题追溯

---

## ⚙️ 配置选项

### 修改默认密码

编辑 `config.py`：
```python
AUTH_PASSWORD = 'your_strong_password'
```

### 调整监控刷新频率

编辑 `client_visual.py`，找到 `_monitor_loop` 方法：
```python
time.sleep(3)  # 改为其他秒数
```

### 自定义主题颜色

编辑 `gui_theme.py`：
```python
COLORS = {
    'bg_dark': '#1e1e1e',       # 深色背景
    'fg_primary': '#00ff00',    # 主文字颜色
    # ... 修改其他颜色
}
```

## 🔒 安全特性

1. **密码哈希验证**
   - 使用SHA256哈希
   - 不传输明文密码

2. **授权确认**
   - 每次连接需服务端授权
   - 显示请求来源IP和端口

3. **可视化警告**
   - 连接时状态栏变红
   - 显示"⚠️ SYSTEM UNDER REMOTE CONTROL"

4. **操作日志**
   - 完整记录所有操作
   - 可导出审计

5. **沙箱环境**
   - 文件操作限制在安全目录
   - 命令白名单机制

## 🛠️ 故障排除

### ❌ 连接失败

**原因**：
- IP地址错误
- 防火墙阻止
- 密码错误

**解决**：
1. ping目标IP确认连通
2. 检查防火墙（允许端口9999）
3. 验证密码

### ❌ 功能无响应

**原因**：
- 网络中断
- 服务端崩溃

**解决**：
1. 查看操作历史中的错误
2. 重新连接
3. 重启服务端

### ❌ 屏幕流卡顿

**解决**：
1. 降低质量滑块（30-40）
2. 检查网络带宽
3. 使用截图代替实时流

详细排查：[VISUALIZATION_GUIDE.md](VISUALIZATION_GUIDE.md)
